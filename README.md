# tegg app

[Hacker News](https://news.ycombinator.com/) showcase using [tegg HTTPController](https://eggjs.org/basics/controller)

## QuickStart

### Development

```bash
pnpm i
pnpm run dev
open http://localhost:7001/
```

### Deploy

```bash
pnpm run build
pnpm start
```

### pnpm Scripts

- Use `pnpm run lint` to check code style
- Use `pnpm test` to run unit test
- se `pnpm run clean` to clean compiled js at development mode once

### Requirement

- Node.js >= 18.x
- Typescript >= 5.x
